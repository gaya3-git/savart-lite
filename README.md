# Savart Lite

A console-based Financial Health Checkup & Advisory Engine built using Python.

## Features

- Savings Ratio Calculator
- Emergency Fund Calculator
- Insurance Adequacy Calculator
- Financial Health Score
- Personalized Recommendations
- Text Report Generation
- Pie Chart Generation
- Bar Chart Generation
- Automated Tests

## Technologies

- Python 3
- Matplotlib
- JSON
- Git
- GitHub

## Installation

Clone the repository:

```bash
git clone https://github.com/gaya3-git/savart-lite.git
```

Go to the project folder:

```bash
cd savart-lite
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## Run the Project

```bash
python main.py
```
